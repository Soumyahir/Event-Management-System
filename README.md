Event Management System

Overview
The Event Management System is a web-based platform that provides all event-related services in one place. Users can organize weddings, corporate events, birthdays, and other occasions by accessing services such as venue booking, decoration, catering, makeup, and photography.

Features
- User Registration and Login
- Event Booking
- Venue Management
- Decoration Services
- Catering Services
- Makeup Services
- Admin Dashboard
- User Dashboard

Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

Future Enhancements
- Online Payments
- AI-based Recommendations
- Real-time Notifications
